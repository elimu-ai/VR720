全景图打开失败怎么办？
您可以先使用其他图片或视频APP打开全景图试试，如果其他APP都可以打开就我们的APP失败，那么麻烦您向我们反馈这个问题。
全景图打开暂时不支持这种图像格式怎么办？
APP并不能保证可以打开任何格式的文件，如果您遇到文件格式不支持的错误时，我们推荐您使用其他处理软件将图像转为 JPEG 格式，或是将视频转为 MP4 格式。
图像过大怎么办？
APP并不能打开特别大的图像（65536x32768），因为这么大的图像会非常消耗手机性能，加载耗时特别长，但是又超过手机分辨率对浏览没有太大帮助。
因此我们建议，您可以使用PS等图像处理软件，缩放图像至16384x8192左右（推荐保存为JPEG格式），这个大小的图像既有比较高的清晰度，又可以很快加载。
全景图为什么有接缝？
请检查您的全景图是否是正常的720度全景（宽高比为2：1），也有可能是您在全景拼接时没有正确对齐。
为什么一个本地APP要网络权限？
APP为是了检查软件有没有更新，以及为了发送错误报告、反馈等为了帮助改进软件功能的操作可以运行， 所以才申请了网络权限。
我们承诺不会收集您的个人隐私或个人文件。您也可以在手机系统管家中禁用本软件的联网功能， 这并不会影响到APP的使用。
如何备份APP设置、全景图数据和分类收藏夹数据？
您可以在设置中点击“备份设置数据至文件”将当前数据以及软件设置保存备份，之后您在卸载重新安装APP之后可以点击“从文件备份恢复设置数据”来快速恢复之前的数据。
但是注意，此备份仅仅备份文件路径，并不会备份文件内容，如果您把数据迁移到新设备上，您同时也需要把全景文件也重新复制，否则会找不到路径。
其他提示
APP为了加快全景读取的速度，默认会开启缓存，这可能会导致占用您的存储空间，您可以选择在设置中关闭（但这可能会导致加载时间变长）， 或是在设置中手动清理缓存以释放存储空间。
注意：由于软件数据全部保存在本地，因此，如果您卸载软件或是在系统应用管理中清除本APP数据这会导致您之前导入的图像数据和保存的分类收藏夹数据全部丢失！ 因此，您可以在设置中导出您的数据至文件保存，如果您需要恢复之前的数据，只需导入之前备份的文件即可。
What if the panorama fails to open?
You can use other picture or video apps to open the panorama first. If all other apps can be opened, our app fails, please give us feedback about this problem.
What if this image format is not supported when panorama is opened?
App does not guarantee that you can open any file format. If you encounter an error that the file format does not support, we recommend that you use other processing software to convert the image to JPEG format, or convert the video to MP4 format.
What if the image is too large?
App can't open a very large image (65536x32768), because such a large image will consume the performance of the mobile phone and take a long time to load, but it is beyond the resolution of the mobile phone, which is not very helpful for browsing.
therefore, we suggest that you can use PS and other image processing software to zoom the image to about 16384x8192 (it is recommended to save it in JPEG format). This size of image has high definition and can be loaded quickly.
Why do panoramas have seams?
Please check whether your panorama is a normal 720 degree panorama (the aspect ratio is 2:1), or you may not have the correct alignment when splicing the panorama.
Why does a local app need network permissions?
In order to check whether the software has been updated, and to send error reports, feedback and other operations to help improve the function of the software, the app can be run, That's why I applied for network permission.
we promise that we will not collect your privacy or personal files. you can also disable the networking function of the software in the mobile system manager, This will not affect the use of app.
How to back up app settings, panorama data and classified favorite data?
You can click "back up setting data to file" in the settings to save and back up the current data and software settings. After you uninstall and re install app, you can click "restore setting data from file backup" to quickly restore the previous data.
but note that this backup only backs up the file path, not the file content. If you migrate the data to a new device, you also need to copy the panoramic file again, otherwise you will not find the path.
Other tips
In order to speed up the panoramic reading, app will turn on the cache by default, which may occupy your storage space. You can choose to turn it off in the settings (but this may lead to longer loading time), Or manually clean up the cache in settings to free up storage space.
Note: since all the software data are saved locally, if you uninstall the software or clear the app data in the system application management, all the image data you imported and the saved classified favorite data will be lost! Therefore, you can export your data to a file to save in the settings. If you need to restore the previous data, you only need to import the previously backed up file.
